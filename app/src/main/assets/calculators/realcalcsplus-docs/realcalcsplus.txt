Real Calcs KML Scripts for Emu48+
Version 3.0

By Eric Rechlin
eric@hpcalc.org
http://www.hpcalc.org

This is a set of HP 39g+, HP 39gs, HP 40gs, HP 48gII, HP 49g+, and HP 50g
"skins" for use with Emu48+ 1.59 or later.  These skins were designed to
make the on-screen calculators look as real as possible.

All of the bitmaps are based on photos by Hewlett-Packard, with the exception
of the 49g+, which was scanned by Joe Horn.  All KML scripts were entirely
written by me, as well as the KMI keyboard layout include files.  The icon
files were created by me as well.

All bitmaps support transparency when used with the "borderless" mode of
Emu48, enabled when both the title bar and menu bar are turned off in the
settings.

All of the skins have touch-sensitive screens for the softkey menus, and all
but the 39/40 have NXT and PREV buttons to the right and left, respectively,
of the softkey menu.  All skins also have full contrast support.


-------------------------------------------------------------------------------


To install a skin, simply extract the necessary files to the Emu48
directory.  The following skins are included, grouped by calculator:

HP 39g+
=======

ROM file required:
	* rom.39g

Keyboard layout file:
	* keyb3940.kmi (see keyboard-layout-3940.png for mappings to a US keyboard)

Icon:
	* 39gp.ico

Variants:

Medium Cropped:
	* 2x display with cropped bezel
	* For resolutions of at least 600 lines
	* Bitmap: real39gp-mc.png
	* KML: real39gp-mc.kml

Medium Full:
	* 2x display with full bezel
	* For resolutions of at least 768 lines
	* Bitmap: real39gp-m.png
	* KML: real39gp-m.kml

Large Cropped:
	* 3x display with cropped bezel
	* For resolutions of at least 1024 lines
	* Bitmap: real39gp-lc.png
	* KML: real39gp-lc.kml

Large Full:
	* 3x display with full bezel
	* For resolutions of at least 1200 lines
	* Bitmap: real39gp-l.png
	* KML: real39gp-l.kml

-------------------------------------------------------------------------------


The keyboard mapping layout images documenting the functionality of each of the
KMI keyboard layout definition files were created with the following web site:

http://www.keyboard-layout-editor.com/

The JSON source files are included if you want to create your own images to
document your layouts if you choose to modify the KMI files.  Also included is
the file keyboard-layout-scancodes.png to show the Emu48 scancodes associated
with each supported key on a US keyboard.
