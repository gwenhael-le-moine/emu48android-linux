Real Calcs KML Scripts for Emu48
Version 3.0

By Eric Rechlin
eric@hpcalc.org
http://www.hpcalc.org

This is a set of HP 38G, HP 39G, HP 40G, HP 48SX, HP 48GX, and HP 49G "skins"
for use with Emu48 1.61 (or Emu48+ 1.59) or later.  These skins were
designed to make the on-screen calculators look as real as possible.

All of the bitmaps are based on photos by Hewlett-Packard, with the exception
of the 39G/40G, which was scanned by me.  All KML scripts were entirely
written by me as well, as well as the KMI keyboard layout include files
(except for keyb38-dm.kmi, described later). The icon files were created by me
as well.

All bitmaps support transparency when used with the "borderless" mode of
Emu48, enabled when both the title bar and menu bar are turned off in the
settings.

All of the skins have touch-sensitive screens for the softkey menus, and all
but the 38G/39G/40G have NXT and PREV buttons to the right and left,
respectively, of the softkey menu.  All skins also have full contrast support.

An older version of the "Medium (Cropped)" 49G script was included with HP's
HP49 SDK for Emu49 and YorkeM and an older version of the "Medium (Full)" 49G
script was included with the standalone YorkeM program.


-------------------------------------------------------------------------------


To install a skin, simply extract the necessary files to the Emu48
directory.  The following skins are included, grouped by calculator:

HP 38G
======

ROM file required:
	* rom.38g

Keyboard layout file:
	* keyb38.kmi (see keyboard-layout-38.png for mappings to a US keyboard)

Alternate keyboard layout file:
	* keyb38-dm.kmi (by Detlef Müller, rename to keyb38.kmi to use)

Icon:
	* 38g.ico

Variants:

Medium Cropped:
	* 2x display with cropped bezel
	* For resolutions of at least 600 lines
	* Bitmap: real38g-mc.png
	* KML: real38g-mc.kml

Medium Full:
	* 2x display with full bezel
	* For resolutions of at least 768 lines
	* Bitmap: real38g-m.png
	* KML: real38g-m.kml

Large Cropped:
	* 3x display with cropped bezel
	* For resolutions of at least 1024 lines
	* Bitmap: real38g-lc.png
	* KML: real38g-lc.kml

Large Full:
	* 3x display with full bezel
	* For resolutions of at least 1200 lines
	* Bitmap: real38g-l.png
	* KML: real38g-l.kml

-------------------------------------------------------------------------------


The keyboard mapping layout images documenting the functionality of each of the
KMI keyboard layout definition files were created with the following web site:

http://www.keyboard-layout-editor.com/

The JSON source files are included if you want to create your own images to
document your layouts if you choose to modify the KMI files.  Also included is
the file keyboard-layout-scancodes.png to show the Emu48 scancodes associated
with each supported key on a US keyboard.
